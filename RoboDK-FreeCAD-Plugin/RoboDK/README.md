# FreeCAD RoboDK Workbench
A FreeCAD Workbench for RoboDK.  

# Description
This workbench allows to export models, curves and points from FreeCAD to RoboDK.

## Features

* Models export.  
* Curves export.  
* Points export. 

## Requirements
* FreeCAD v1.X.Y  
* Python3  
* Qt5
